# Dumps

Dieser Ordner enthält vollständige oder partielle Dumps aus Heimgewebe-Repos.

Regeln:
- Dumps werden nicht verändert.
- Keine Ergänzungen oder Korrekturen im Dump selbst.
- Analyse erfolgt immer außerhalb oder darauf referenzierend.

KI-Agenten:
- Dumps vollständig lesen.
- Keine Lücken interpolieren.
- Fakten strikt von Interpretation trennen.
