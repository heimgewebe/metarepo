# ADR-0023 Epistemic Autopoiesis
![status: accepted](https://img.shields.io/badge/status-accepted-green)
Datum: 2025-12-19
Status: Accepted
Owner: epistemic-team

## Kontext

Das Heimgewebe hat sich von einem technischen Orchestrierungssystem
zu einem erkenntnisverarbeitenden System entwickelt.

Bisher fehlte eine explizite Verfassung für:
– Entscheidungen
– Unsicherheiten
– Widersprüche
– Begriffsdrift

Dies führte zu impliziten Annahmen und nicht rekonstruierbaren Denkpfaden.

## Entscheidung

Das Heimgewebe verpflichtet sich auf das Prinzip der
**epistemischen Autopoiesis**.

Erkenntnisrelevante Vorgänge müssen:
– explizit
– rekonstruierbar
– unsicherheitsfähig
sein.

## Konsequenzen

– decision.preimage wird konzeptionell verpflichtend
– Unsicherheit wird ein First-Class Object
– Widerspruch wird ein legitimes Event
– Begriffe unterliegen Beobachtung statt Fixierung

Diese ADR erzwingt **noch keine technische Umsetzung**,
sondern setzt einen verbindlichen Orientierungsrahmen.

## Risiken

– erhöhte Komplexität
– anfängliche Reibung
– menschliche Abkürzungstendenzen

Diese Risiken werden bewusst akzeptiert.

## Alternativen

– implizite Entscheidungen
– rein pragmatische Orchestrierung
– Wahrheitsfixierung

Diese Alternativen wurden verworfen, da sie langfristig
Systemblindheit begünstigen.
